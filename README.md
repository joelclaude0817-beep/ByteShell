# ByteShell
Hello, Let me tell you how i made a shell that can access to any commands in any operating systems.

## Cloning The Repository
```bash
git clone https://github.com/joelclaude0817-beep/ByteShell/
```

## Changing Directorys
```bash
cd ByteShell
```

## Running The Shell
```bash
gcc -o byteshell byteshell.c
./byteshell
```

And That's How you do it.
